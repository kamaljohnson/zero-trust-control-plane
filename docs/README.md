# Documentation

Documentation for the Zero Trust Control Plane lives in the **Docusaurus docs site**.

- **To view the docs**: Run `npm install && npm run start` in the [`docs-site/`](../docs-site/) directory (default: http://localhost:3001).
- The frontend "Docs" button links to this site; set `NEXT_PUBLIC_DOCS_URL` in the frontend environment (e.g. `http://localhost:3001` in development).

See [docs-site/README.md](../docs-site/README.md) for details.
